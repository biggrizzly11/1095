Instructions for Filling out form 1094-C

  • Name - Enter full name
  • Address - Enter full address, make sure to include the apartment no.
  • City - Enter applicable city
  • State - Enter applicable state
  • Zip - Enter applicable zip
  • Phone - Enter phone with dashes
  • Social Security Number - Enter SSN with dashes
  • Health plan start month - Enter a 2-digit number for start month. For example if the plan started in January, enter 01.
  • Offer of Coverage - Enter one of the following codes "1A, 1B, 1C, 1D, 1E, 1F, 1G, 1H, 1I" for all applicable boxes
  • Employee Share - Enter the share of lowest cost monthly premium
  • Applicable Section 4980H Safe Harbor - Employee offer and coverage of section 4980H Safe Harbor Code. The code must be entered as one of the following "2A, 2B, 2C, 2D, 2E, 2F, 2G, 2HI"

If Employer provided self-insured coverage

  • Name - Enter full name
  • Social Security Number - Enter SSN with dashes
  • Date of Birth - Enter if SSN was not avaliable
  • Covered Months - Enter an "X" for all applicable fields
